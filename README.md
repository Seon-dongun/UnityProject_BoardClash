# UnityProject_BoardClash
- Unity 엔진으로 제작한 전략 시뮬레이션 게임 BoardClash 입니다.  
- 전략적으로 아군 캐릭터를 보드판에 배치하여, 전투에서 승리하는 게임입니다.
- 본 게임은 쿡앱스 게임 클라이언트 과제 전형 프로젝트로 제작한 게임이며, 해당 게임 리소스 파일은 쿡앱스로부터 제공받아 사용하였습니다.
